# Atliq-sql-ad-hoc-analysis
AtliQ Hardware – SQL Data Analyst Project

## Welcome to this project repository!
This project is part of the Codebasics SQL Challenge, designed to simulate a real-world business scenario for a Data Analyst role.

## Company Background

AtliQ Hardwares is a leading computer hardware company in India with a growing global presence.

However, the management lacks data-driven insights, making it difficult to take fast and informed decisions.

## Problem Statement

To improve decision-making, the Data Analytics Director launched a challenge to evaluate candidates on:

SQL skills for data analysis
Ability to convert data into business insights
Communication and storytelling for management

## My Role

As a Data Analyst applicant, I performed:

📄 Solved 10 ad-hoc business questions using SQL

🧠 Extracted meaningful insights from raw data

📊 Built a Power BI dashboard for executives

📽️ Created a presentation to explain insights clearly

## Project Structure
📄 ad-hoc-requests.pdf → Business problems

🧠 SQL_Queries.sql → All SQL solutions

📽️ Presentation.pptx → Insights storytelling

## Tools & Skills Used

SQL → Data extraction & analysis

Power BI → Dashboard & visualization

Excel → Data cleaning & validation

Data Storytelling → Business insights presentation

## Key Insights

Identified top-performing products by division

Analyzed sales trends over time

Found high-value customers and discount patterns

Compared manufacturing costs across products

Evaluated channel-wise sales contribution
